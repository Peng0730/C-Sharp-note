# 继承的特点
* 子类继承了父类的属性和方法但没有继承私有字段和构造函数
* 当新建一个子类的对象时，会先执行父类的空参的构造函数，在子类的内部创建一个父类的对象
* 如果父类和子类中存在同名的方法，程序会执行子类的方法，而不会执行父类的，父类的方法会被隐藏
如果故意隐藏父类的方法或者成员，可在方法返回值前加new  不是故意隐藏的程序会报错
* 故意隐藏构造函数的new语法
``` c#
public new void study() //构造方法
{

}
```
### new的作用
* 创建一个对象
* 隐藏父类的成员

# 继承的作用
